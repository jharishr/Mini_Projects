# Neural Network from Scratch

Implementing a fully connected neural network from scratch using only Numpy to classify Handwritten digits from the MNIST dataset.

Forward propagation and backpropagation implemented manually

Input Layer : 784 neurons (28x28 flattened image)
Hidden Layer : Fully connected with ReLU activation
Output Layer : 10 neurons with Softmax activation
Optimization : Gradient descent

# How to run: 
```bash
pip install -r requirements.txt
python nn.ipynb
